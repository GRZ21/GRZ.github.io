JavaWeb中的HTML

本文最后更新于：11 天前

HTML和CSS，还有JS都属于前端的范畴，SMBMS系统因为看不懂前端请求搞得流产了……只能回头再看前端了。HTML中重点在表单那块。

HTML简介：

HTML称为超文本标记语言，是一种标识性的语言。它包括一系列标签．通过这些标签可以将网络上的文档格式统一，使分散的Internet资源连接为一个逻辑整体。HTML文本是由HTML命令组成的描述性文本，HTML命令可以说明文字，图形、动画、声音、表格、链接等。（百度百科）

HTML中的基础标签：

标题标签：

这个类似于Markdown语法中的标题标签。

段落标签：

分段用，html语言不会自动分段。

换行标签：

这个和段落标签类似，但是区别在于行间距的大小。段落标签比换行标签大。

水平线标签：

这个就是加一条水平分割线。

字体标签：

这个就是调整字体样式。

特殊符号：

图像标签：

这个就是image标签。但是需要填写文件路径。

注意绝对路径和相对路径的区别。

多媒体标签：

多媒体也就两种，音频和视频。

链接标签：

这个就是实现链接跳转的功能。

iframe标签：

这个类似于在一个网页中再嵌套一个网页。使用target可以搭配起来用。

表格标签：

创建表格的标签。

列表标签：

这个分为三种：有序列表，无序列表，自定义列表。

form表单（重点）：

这个是HTML中的重点。form表单里包含许多不同类型的标签，大部分是使用input type = “”什么类型实现。

属性 说明
type 指定元素的类型：text、password、checkbox、radio、submit、reset、file、hidden、image和button，默认为text
name 指定表单元素的名称
value 元素的初始值。type为radio时必须指定一个值
size 指定表单元素的初始宽度。当type为text或者password时，表单元素的大小以字符为单位。对于其他的类型，宽度以像素为单位。
maxlength type为text或者password时，输入的最大的字符数
checked type为radio或者checkbox时候，指定的按钮是否被选中

页面结构分析：

元素名 描述
header 标题头部区域的内容
footer 标题的脚部的内容
section web页面中的一块独立区域
article 独立的文章内容
aside 相关内容或应用
nav 导航类辅助内容
